```
/
├── Главная (/)                                 → как на HH.ru: большой поиск + топ-стажировки + категории
│
├── Поиск стажировок (/search)                  → результаты поиска + фильтры слева (основная страница поиска)
│   ├── Детальная страница стажировки (/internship/:id)   → аналог вакансии на HH
│   └── ?query=...&filters=...                  → динамические URL с параметрами
│
├── Для студентов
│   ├── Регистрация / авторизация (/auth/student)
│   ├── Личный кабинет студента (/my)           → дашборд
│   │   ├── Профиль (/my/profile)
│   │   ├── Резюме (/my/resume)                 → редактор резюме + несколько версий
│   │   ├── Мои отклики (/my/applications)
│   │   ├── Избранное (/my/favorites)
│   │   ├── Рекомендации (/my/recommendations)
│   │   ├── Уведомления (/my/notifications)
│   │   └── Чат/сообщения (/my/messages)
│   └── Публичный профиль студента (/student/:id)   → видимый компаниям (если разрешено)
│
├── Для компаний / работодателей
│   ├── Регистрация / авторизация (/auth/company)
│   ├── Личный кабинет компании (/employer)     → дашборд
│   │   ├── Профиль компании (/employer/profile)
│   │   ├── Мои стажировки (/employer/internships)
│   │   │   ├── Создать стажировку (/employer/internships/new)
│   │   │   └── Редактировать (/employer/internships/:id/edit)
│   │   ├── Отклики и кандидаты (/employer/responses)
│   │   ├── Аналитика и статистика (/employer/analytics)   → премиум
│   │   ├── Чат/сообщения (/employer/messages)
│   │   └── Продвижение стажировок (/employer/promotion)
│   └── Публичная страница компании (/company/:id)   → брендированная витрина + список стажировок
│
├── Информационные / статические страницы
│   ├── О проекте(/about)
│   ├── Политика конфиденциальности (/about/privacy)
│   ├── Для студентов (/for-students)           → гайды, советы
│   ├── Для компаний (/for-employers)
│   ├── Блог / Статьи (/articles)               → "Как составить резюме", "Топ стажировок 2026"  НЕ самое обязательное
│   ├── Контакты (/contacts)
│   ├── Правила размещения (/rules)
│   └── FAQ (/faq)
│
└── Админка (закрытая) (/admin)                 → только для модераторов
    ├── Проверка стажировок
    ├── Проверка профилей
    ├── Жалобы и спам
    └── Статистика платформы
```


Студент → поиск и отклик

Главная → ввод запроса ("frontend стажировка Москва удалёнка") → /search
Фильтры (левый сайдбар) → применить → список карточек
Клик по карточке → /internship/12345
"Откликнуться" → если резюме заполнено → отправка + статус в /my/applications
Если нет резюме → редирект на /my/resume с напоминанием

Компания → размещение стажировки

/auth/company → регистрация + верификация (ИНН / email компании)
/employer → дашборд
"Разместить стажировку" → форма (очень похожа на HH):
Название стажировки
Описание (обязанности, что получит студент)
Требования (курс, навыки, вузы)
Условия (оплата/стипендия, длительность, формат, локация)
Навыки (теги)
Контактное лицо

Публикация → модерация (1–24 ч) → появляется в поиске

Общий поисковый движок

Elasticsearch / OpenSearch
Основные поля для индексации: название, описание, навыки, локация, оплата, длительность, компания, дата публикации
Автодополнение запросов (как на HH)
Персональные рекомендации (для залогиненных студентов) — на основе навыков / резюме / предыдущих откликов

Проверка подлинности компании:

Чтобы подтвердить, что вы — это реально та компания, чей ИНН вы указали, нужно загрузить сканы документов:

Для юрлиц: Свидетельство о регистрации (ОГРН), лист записи ЕГРЮЛ или Свидетельство о постановке на налоговый учет.

Для ИП: Аналогичные документы (Лист записи ЕГРИП).

Для частных лиц/рекрутеров: Фотография паспорта.

Без этих бумаг регистрация не будет завершена, и вы не сможете публиковать вакансии в полном объеме.
